Ce LAB met l’accent sur les éléments suivants :
 Sécurité de base d’un équipement réseau
-  Mots de passe sécurisés au niveau d’un routeur
-  Configuration avancée du protocole SSH
-  Protection contre les attaques par force brute
-  Désactivation des ports inutilisés d’un commutateur
-  Sécurité des ports d’un commutateur
 Gestion des fichiers d’un équipement réseau
-  Récupération du mot de passe d’un équipement réseau
-  Sauvegarde et restauration des fichiers d’un équipement réseau
Configuration AAA
- Configuration AAA locale
-  Configuration AAA basée sur un serveur Radius
-  Configuration AAA basée sur un serveur TACACS+
Configuration des listes de contrôle d’accès
-  Configuration d’une ACL standard
-  Configuration d’une ACL étendue
 Configuration de la sécurité des réseaux sans fil
-  Configuration d’un Wi-Fi
-  Configuration de la sécurité d’un Wi-Fi avec Radius
Contexte / Scénario
Le but de ce LAB est de configurer la topologie pour pouvoir s’habituer avec les différentes notions de la sécurités des réseaux Cisco.
